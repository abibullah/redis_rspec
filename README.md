# RedisRspec

  This gem will create a redis-server for rspec on port 6666

## Installation

Add this line to your application's Gemfile:

    gem 'redis_rspec'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install redis_rspec

## Add this to the spec_helper
  
  require 'redis_rspec'
